## Введение

SessiaAval - это система управления салоном красоты, которая позволяет:
- Клиентам записываться на услуги и управлять своим балансом
- Мастерам просматривать расписание и завершать услуги
- Модераторам управлять услугами и мастерами
- Администраторам управлять пользователями системы

### Системные требования
- Операционная система: Windows 10/11, Linux, macOS
- .NET 9.0 Runtime
- Подключение к интернету (для работы с базой данных)

## Начало работы

### Запуск приложения

1. Запустите файл `SessiaAval.exe` (Windows) или выполните `dotnet run` в папке проекта
2. Откроется окно авторизации

### Регистрация нового пользователя

1. На экране входа нажмите кнопку **"РЕГИСТРАЦИЯ"**
2. Заполните обязательные поля:
   - **ФИО**: Введите полное имя (например: Иван Иванов)
   - **Email**: Введите уникальный email адрес
   - **Пароль**: Придумайте пароль (минимум 3 символа)
   - **Телефон**: Введите номер телефона в формате +7XXXXXXXXXX
3. Нажмите кнопку **"ЗАРЕГИСТРИРОВАТЬСЯ"**
4. При успешной регистрации откроется личный кабинет пользователя

**Индикация процесса:**
- ✓ Зеленое сообщение "Регистрация успешна"
- ✗ Красное сообщение об ошибке, если email уже существует

### Вход в систему

1. Введите **Email** и **Пароль**
2. Нажмите кнопку **"ВОЙТИ"**
3. Система автоматически определит вашу роль и откроет соответствующий личный кабинет

**Тестовые аккаунты:**
- Пользователь: user1@salon.com / 123
- Мастер: master1@salon.com / 123
- Модератор: moderator@salon.com / 123
- Администратор: admin@salon.com / 123

## Функции для пользователя (клиента)

### 1. Просмотр и пополнение баланса

#### Просмотр баланса
1. В боковом меню нажмите **"МОЙ БАЛАНС"**
2. В верхней части экрана отображается текущий баланс
3. Ниже показана таблица истории транзакций с колонками:
   - Дата
   - Тип операции (Пополнение / Оплата услуги)
   - Сумма
   - Последние 4 цифры карты (для пополнений)

#### Пополнение баланса
1. Нажмите кнопку **"ПОПОЛНИТЬ БАЛАНС"**
2. В открывшемся окне заполните данные карты:
   - **Номер карты**: 16 цифр (например: 1234567812345678)
   - **Срок действия**: Формат MM/YY (например: 12/25)
   - **CVV**: 3 цифры (например: 123)
   - **Сумма**: Введите сумму пополнения (больше 0)
3. Нажмите **"ПОПОЛНИТЬ"**

**Индикация процесса:**
- ✓ Баланс обновляется мгновенно
- ✓ Создается запись в истории транзакций
- ✗ Ошибка валидации, если данные введены неверно

**Валидация:**
- Номер карты должен содержать ровно 16 цифр
- Срок действия в формате MM/YY, месяц от 01 до 12
- CVV должен содержать ровно 3 цифры
- Сумма должна быть больше 0

### 2. Запись на услугу

1. В боковом меню нажмите **"МОИ ЗАПИСИ"**
2. Нажмите кнопку **"ЗАПИСАТЬСЯ"**
3. В диалоговом окне:
   - **Выберите услугу** из выпадающего списка (показывается название, цена и длительность)
   - **Выберите мастера** из списка (показывается ФИО и уровень квалификации)
   - **Выберите дату** записи (по умолчанию завтра)
   - **Выберите время** из доступных слотов (с 09:00 до 20:30 с шагом 30 минут)
4. Нажмите **"ЗАПИСАТЬСЯ"**

**Индикация процесса:**
- ✓ Показывается диалог с подтверждением
- ✓ Отображается присвоенный номер в очереди
- ✓ Запись появляется в таблице "МОИ ЗАПИСИ"

**Важно:**
- Можно выбрать любого мастера, даже если он занят
- Номер в очереди присваивается автоматически
- Оплата спишется только после завершения услуги мастером

### 3. Просмотр своих записей

1. Перейдите в раздел **"МОИ ЗАПИСИ"**
2. В таблице отображаются все ваши записи:
   - **Дата/Время**: Когда назначена услуга
   - **Услуга**: Название услуги
   - **Мастер**: ФИО мастера
   - **Номер в очереди**: Ваш номер
   - **Статус**: 
     - `pending` - Ожидает выполнения
     - `completed` - Завершена

**Индикация статусов:**
- 🟡 pending - желтый цвет, услуга запланирована
- 🟢 completed - зеленый цвет, услуга выполнена

### 4. Написание отзывов

1. Перейдите в раздел **"ОТЗЫВЫ"**
2. В таблице отображаются завершенные услуги
3. Нажмите **"НАПИСАТЬ ОТЗЫВ"** напротив нужной услуги
4. В диалоговом окне:
   - **Выберите оценку** от 1 до 5 звезд
   - **Напишите комментарий** (необязательно)
5. Нажмите **"ОТПРАВИТЬ"**

**Индикация процесса:**
- ✓ Отзыв сохранен
- ✓ Кнопка "НАПИСАТЬ ОТЗЫВ" становится неактивной после отправки

**Ограничения:**
- Отзыв можно оставить только для завершенных услуг (статус `completed`)
- Один отзыв на одну услугу

### 5. Просмотр каталога услуг

1. Перейдите в раздел **"УСЛУГИ"**
2. Отображается таблица всех доступных услуг:
   - Название услуги
   - Категория
   - Цена
   - Длительность в минутах

**Индикация:**
- Услуги отсортированы по категориям
- Цены отображаются в рублях с копейками

### 6. Выход из системы

1. Нажмите кнопку **"РАЗЛОГИНИТЬСЯ"** в боковом меню
2. Откроется окно входа
3. Приложение не закрывается, можно войти под другим аккаунтом

## Функции для мастера

### 1. Просмотр клиентов и записей

1. После входа автоматически открывается раздел **"МОИ КЛИЕНТЫ"**
2. В верхней части отображается статистика:
   - **Всего записей**: Общее количество
   - **Активные**: Записи со статусом `pending`
   - **Завершено**: Записи со статусом `completed`
3. В таблице показаны все записи:
   - Дата и время
   - ФИО клиента
   - Название услуги
   - Номер в очереди
   - Статус
   - Кнопка действия

**Индикация статусов:**
- 🟡 pending - Кнопка "Завершить" активна
- 🟢 completed - Показывается текст "Готово"

### 2. Завершение услуги

1. В таблице клиентов найдите запись со статусом `pending`
2. Нажмите кнопку **"Завершить"**
3. Система автоматически:
   - Проверит баланс клиента
   - Спишет стоимость услуги
   - Создаст транзакцию оплаты
   - Изменит статус на `completed`

**Индикация процесса:**
- ✓ Показывается диалог с информацией о списании
- ✓ Отображается остаток на балансе клиента
- ✗ Ошибка, если у клиента недостаточно средств

**Важно:**
- Если у клиента недостаточно средств, услуга не завершается
- Статус остается `pending` до пополнения баланса клиентом
- После завершения услугу нельзя отменить

### 3. Просмотр назначенных услуг

1. Нажмите **"МОИ УСЛУГИ"** в боковом меню
2. Отображается список услуг, которые вам назначены:
   - Название услуги
   - Категория
   - Цена
   - Длительность

**Индикация:**
- Показывается общее количество назначенных услуг
- Услуги отсортированы по названию

### 4. Управление квалификацией

1. Нажмите **"КВАЛИФИКАЦИЯ"** в боковом меню
2. Отображается:
   - Текущий уровень квалификации (крупным шрифтом)
   - Специализация (если указана)
   - Кнопка "Подать заявку на повышение"

#### Подача заявки на повышение
1. Нажмите **"Подать заявку на повышение"**
2. Подтвердите действие
3. Заявка отправлена модератору

**Индикация процесса:**
- ✓ Кнопка меняется на "✓ Заявка подана"
- ✓ Показывается текст "Ваша заявка ожидает рассмотрения"
- Кнопка становится неактивной до рассмотрения заявки

### 5. Просмотр отзывов о себе

1. Нажмите **"ОТЗЫВЫ"** в боковом меню
2. В верхней части показывается статистика:
   - Всего отзывов
   - Средняя оценка (от 1 до 5)
3. В таблице отображаются все отзывы:
   - Дата отзыва
   - ФИО клиента
   - Название услуги
   - Оценка (из 5)
   - Комментарий

**Индикация:**
- Если отзывов нет, показывается сообщение с рекомендацией
- Средняя оценка рассчитывается автоматически

## Функции для модератора

### 1. Управление услугами

#### Просмотр услуг
1. После входа открывается раздел **"УСЛУГИ"**
2. Отображается таблица всех услуг салона:
   - Название
   - Категория
   - Цена
   - Длительность
   - Кнопки действий

#### Добавление услуги
1. Нажмите **"ДОБАВИТЬ УСЛУГУ"**
2. Заполните поля:
   - **Название**: Название услуги
   - **Категория**: Выберите из списка
   - **Цена**: Стоимость в рублях
   - **Длительность**: Время в минутах
   - **Описание**: Краткое описание (необязательно)
3. Нажмите **"СОХРАНИТЬ"**

**Индикация:**
- ✓ Услуга добавлена в таблицу
- ✗ Ошибка, если не заполнены обязательные поля

#### Редактирование услуги
1. Нажмите **"Изменить"** напротив услуги
2. Измените нужные поля
3. Нажмите **"СОХРАНИТЬ"**

#### Удаление услуги
1. Нажмите **"Удалить"** напротив услуги
2. Подтвердите удаление
3. Услуга удалена из системы

#### Назначение мастера на услугу
1. Нажмите **"Назначить мастера"** напротив услуги
2. Выберите мастера из списка
3. Подтвердите назначение

**Индикация:**
- ✓ Мастер добавлен к услуге
- Мастер теперь может выполнять эту услугу

#### Отмена назначения
1. Нажмите **"Отменить назначение"** напротив мастера
2. Подтвердите действие
3. Связь удалена

### 2. Управление мастерами

1. Нажмите **"МАСТЕРА"** в боковом меню
2. Отображается таблица всех мастеров:
   - ФИО
   - Уровень квалификации
   - Специализация
   - Индикатор заявки
   - Кнопки действий

#### Одобрение заявки на квалификацию
1. Найдите мастера с индикатором **"Заявка на повышение"**
2. Нажмите **"Одобрить заявку"**
3. Подтвердите действие

**Индикация процесса:**
- ✓ Уровень квалификации увеличен на 1
- ✓ Индикатор заявки исчезает
- ✓ Показывается сообщение об успехе

#### Ручное повышение квалификации
1. Нажмите **"Повысить"** напротив любого мастера
2. Подтвердите действие
3. Уровень квалификации увеличен на 1

## Функции для администратора

### 1. Управление пользователями

1. После входа открывается раздел **"ПОЛЬЗОВАТЕЛИ"**
2. Отображаются все клиенты салона (роль "Пользователь")

2. Заполните данные:
   - ФИО
   - Email (уникальный)

   - Телефон

1. Нажмите **"Изменить"** напротив пользователя
2. Измените нужные поля

   - Количество записей

- ⚠️ Предупреждение о каскадном удалении

### 2. Управление сотрудниками

**Индикация:**
- Для мастеров автоматически создается профиль с квалификацией 1
- Для модераторов создается только учетная запись

## Индикация процессов

### Цветовая индикация статусов

**Записи:**
- 🟡 **pending** (желтый) - Запись ожидает выполнения
- 🟢 **completed** (зеленый) - Услуга завершена

**Кнопки:**
- 🔵 **Синие кнопки** - Основные действия (Записаться, Сохранить, Войти)
- ⚪ **Серые кнопки** - Вторичные действия (Отмена, Назад)
- 🔴 **Красные кнопки** - Опасные действия (Удалить)

### Сообщения системы

**Успешные операции:**
- ✓ Зеленая галочка и текст
- Автоматическое закрытие через 2-3 секунды

**Ошибки:**
- ✗ Красный крестик и описание ошибки
- Требуется нажатие кнопки "OK"

**Предупреждения:**
- ⚠️ Желтый треугольник
- Требуется подтверждение действия

### Индикаторы загрузки

- При загрузке данных из базы показывается анимация
- При сохранении кнопка становится неактивной
- После завершения операции интерфейс обновляется автоматически

## Часто задаваемые вопросы

**В: Что делать, если забыл пароль?**  
О: Обратитесь к администратору системы для сброса пароля.

**В: Можно ли отменить запись?**  
О: В текущей версии отмена записи недоступна. Обратитесь к администратору.

**В: Когда спишутся деньги за услугу?**  
О: Оплата списывается автоматически после того, как мастер завершит услугу.

**В: Что делать, если недостаточно средств?**  
О: Пополните баланс через раздел "МОЙ БАЛАНС". Мастер сможет завершить услугу после пополнения.

**В: Можно ли выбрать мастера, который занят?**  
О: Да, система позволяет записаться к любому мастеру независимо от его загруженности.

**В: Как узнать свой номер в очереди?**  
О: Номер в очереди отображается в разделе "МОИ ЗАПИСИ" и в подтверждении после записи.
